  • NO COUNTRY - Proyecto HealthTech (2024)

    Dentro del marco de iniciativas de simulaciones laborales que promociona No Country, se trabajó en una Web-app para la realización de turnos médicos, establecer horarios con los distintos doctores y, si es necesario, links de videollamadas. Este proyecto se realizó de manera conjunta con programadores de distintas nacionalidades de forma remota aplicando metodología Scrum y las tecnologías para Backend: Java, SpringBoot y en bases de datos; MySql y para el Frontend:TypeScript y React. Este es el repositorio del proyecto y a continuación puedes ver un video presentación del proyecto
